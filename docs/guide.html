❓ 常见问题 FAQ

1. 下载资源没反应/失败？

请尝试关闭 VPN 或代理；如仍无法下载，请切换为阿里云、GitHub 或 Gitee 源。

2. 文件无法打开/报病毒提示？

请关闭 Windows Defender 或添加信任项；如果仍担心安全问题，请直接删除本工具。

3. 无法复制/报错解密失败？

请以管理员身份运行安装器，确保目标文件夹可写且游戏未运行。

4. 安装后仍是俄文？

检查目录是否为指定的客户端，Res_mods 文件夹中 版本文件夹 （如1.35.0.0） 是否存在。

5. 游戏更新后语言丢失？

每次游戏大版本更新后都需重新打补丁，请重新运行安装器。

6. 安装器无法运行？

请确保系统具备 .NET 或 Python 环境，或右键“以管理员身份运行”尝试。

7. 会被封号吗？

不会。本工具仅替换本地语言文件，不含任何 DLL 注入或修改游戏核心内容。

8. 如何恢复原版语言？

运行安装器 → 点击“卸载补丁”按钮即可恢复原始俄文界面。

9. 游戏更新后安装最新版本游戏内仍无法汉化？

确保游戏已更新至最新版本，在游戏关闭的情况下运行安装器然后重新启动游戏。
